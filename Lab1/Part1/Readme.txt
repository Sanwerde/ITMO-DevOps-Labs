FROM debian
Отсутствие версии у базового образа, могут возникнуть конфликты из-за версии.
Исправление: указание конкретной версии.

RUN apt-get update 
RUN apt-get install -y cowsay fortune
Объединение команд RUN без очистки кэша.
Исправление: объединение в один слой и очистка кэша.

ENV PASSWORD=1234567890
Хранение паролей в образе, легко просматриваются.
Исправление: передача через окружение, например, docker run -e PASSWORD="Fgt123kmcbY" cowsayimage 

Плохие практики:
1. Так как по умолчанию процессы внутри контейнера запускаются от имени пользователя root, то злоумышленник получает эти права. И, возможно, он как-то сможет найти уязвимость, которая позволит ему получить доступ к хосту, где он будет с правами root.
2. Запись каких-то важных данных в контейнер. Они временны, поэтому возможна потеря данных.
